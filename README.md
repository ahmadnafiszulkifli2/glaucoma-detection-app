# Glaucoma Detection App Using ResNet-50 and Grad-CAM XAI

## Project Overview
This project presents a deep learning-based solution for detecting Glaucomatous Optic Neuropathy (GON) from digital fundus images. The model uses ResNet-50 with transfer learning for binary classification between GON+ and healthy images. Grad-CAM XAI is applied to improve interpretability by highlighting image regions influencing the model's prediction.

## Objectives
- Detect glaucoma from fundus images
- Reduce black-box behaviour using Grad-CAM
- Improve reliability through image preprocessing
- Develop a practical MATLAB-based application
